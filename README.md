# Beyond Limits Bootcamp

Membership platform for **beyondlimitsbootcamp.com** — a premium performance & lifestyle brand
for high-performing professionals.

**Isolated project:** its own repo, Vercel project, and database. It does **not** share code,
deploys, or data with Krystalore or any other site.

## Status
Phase 1 (Foundation) — in design. See
[`docs/superpowers/specs/2026-07-15-beyond-limits-foundation-design.md`](docs/superpowers/specs/2026-07-15-beyond-limits-foundation-design.md).

### Roadmap
1. **Foundation** — marketing site, auth, 5 membership tiers, admin console, dashboard, impersonation.
2. **Members & Video** — gated live stream + VOD library.
3. **Monetization** — GoHighLevel checkout, coupons, affiliate `/shop`, white-label landers.
4. **Engagement** — chat→email, CRM, community, habit tracker, broadcast.
